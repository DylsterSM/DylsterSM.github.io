Hello World!

Above, an h1 element has a font-size styling of 300%! H1 already signifies a large heading of text, but adding that styling allows the text to get to a larger size that we want.

Moved!

The above "p" element has a padding of 30px to push the text over. This allows better page formatting and element placement!


Brown!


Above, there is a p tag within a div tag, the div tag has a background-color styling of brown! Background-colors can make an otherwise fully white page more interesting with other colors!


Finally, all elements have a styling of the color white and the text-decoration "overline." Coloring text can help distinguish more important text to a user as well as just making a more interesting page to look at. Overline allows for some extra text seperation!
